# Site para gestão de equipes em paróquias, com automação e disparos via telegram
Este é o site oficial dos acólitos da comunidade. Ele serve como uma plataforma para cadastro de membros, divulgação de nossas redes sociais e visualização dos aniversariantes do dia, além disso no back existem automações de disparos via telegram.

## Funcionalidades

- **Cadastro de membros:** Permite que novos usuários se registrem na comunidade.
- **Comunidade:** Apresenta links para nossas redes sociais e canais de comunicação.
- **Aniversariantes do dia:** Lista quem está comemorando aniversário hoje.

## Tecnologias Utilizadas

- React, Typescript, Tailwind

O site está disponível em `(https://acolitosnsc.tinkermin.com.br)`
